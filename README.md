# AI-Powered Customer Churn Prediction System 🔥

## Overview

This project is an AI-powered Customer Churn Prediction System developed using Machine Learning and Streamlit. The system predicts whether a customer is likely to churn based on customer behavior, subscription details, and service usage patterns.

---

## Features

* Real-time customer churn prediction
* Interactive Streamlit dashboard
* Machine learning model using XGBoost
* Analytics and visualization dashboard
* Retention recommendations
* Professional UI design
